
# Whatsapp clone
Whatsapp clone app using **React Native with Expo 50** and Clerk for authentication using OTP (One time password)

## Built With
* **Expo 50.0.7** 
* **React 18.2.0**
* **Typescript 5.1.3** 
* **Clerk 0.20.5** 
* **Expo Router 3.4.7**
